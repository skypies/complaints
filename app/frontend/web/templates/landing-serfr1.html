Hello ! You have requested the URL complaints.serfr1.org, but this has been stale for years now.

The new URL is stop.jetnoise.net. Please update your bookmarks, and any 'add to homescreen' links you might have, to use stop.jetnoise.net.

I've kept the old URL alive all this time, but recently discovered that using it was the cause of a pernicious problem for some users, where from time to time "the site makes me login for every complaint".

(The issue ? A cookie that was used during the login exchange, which was issued for stop.jetnoise.net, but in response to a request sent to complaints.sefr1.or. Some browsers, notably Safari, considered this kind of cross-site cookie a possible privacy violation, and so would drop it. Sometimes.)
